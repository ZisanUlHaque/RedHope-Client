-BloodHope is a web app that helps people find blood donors easily. The goal is to make blood donation simple and save lives. BloodHope provides a platform where users can register as donors, find donors nearby, and help save lives.

Features

-Register as a donor or user -Search for blood donors by blood group and location -Request blood from available donors -Admin panel to manage users and requests -Works on both mobile and desktop

Tech I used Used

-Frontend: React, Tailwind CSS -Backend: Node.js, Express.js -Database: Firebase Firestore -Authentication: Firebase Auth -Hosting: Vercel
